## Project Name
ToDoList

## Project Description
This android app helps you remember your tasks by storing them on you mobile phone

## Project Features
Create a user account<br>
All the data is stored in a remote database (Firebase)<br>
Easy to use UI<br>

## Prerequisites
Android Oreo
